# ai-helpdesk-prototype

## Configuración local

1. Abrir CMD y moverse a la carpeta ai:

```bash
cd C:\xampp\htdocs\ai-helpdesk-prototype\ai
```

2. Crear el modelo en Ollama:

```bash
ollama create mesa_ayuda_municipio -f Modelfile
```

3. Instalar dependencias Python:

```bash
pip install -r requirements.txt
```

4. Ejecutar aplicación local:

```bash
Abrir XAMPP

Acceder a http://localhost/ai-helpdesk-prototype/public/index.php

Escribir un ticket y enviar
```
